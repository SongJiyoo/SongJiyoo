## Hi there 🧸

I design and build backend systems for real-world applications, with a strong focus on scalability, clean architecture, and long-term evolution.

Currently working on a production-oriented e-commerce platform developed as a real product. The system includes backend, web and mobile (Android) applications, with emphasis on performance, maintainability, and with focus on scalability (caching, async processing, event-driven flows), reducing latency and improving system performance.

I’m particularly interested in system design and software architecture, and aim to build systems end-to-end from backend logic to user-facing experiences.

Designing systems with cloud-native architecture in mind, including automated CI/CD pipelines (GitHub Actions) for build, testing, and deployment.

- Mobile: Kotlin, Jetpack Compose
- Frontend: React (fundamentals), REST/GraphQL integration
